# Role Prompt: Quality, Risk, and Enablement Plan

- Role: Development Manager
- Status: draft
- Owner: engineering-management
- Version: 1.0.0
- Last Updated: 2026-02-18

## Prompt Text

```text
Act as a development manager running an AI-assisted team.
Create a 30-day plan that covers:
- team enablement and training priorities
- review capacity and quality risks
- CI gate adoption status
- escalation triggers and ownership
Return:
1) weekly execution plan
2) risk register (severity + owner)
3) KPI dashboard definition
4) decisions required from leadership
```
